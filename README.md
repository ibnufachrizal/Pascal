# Code Pascal

## Usage
  ```bash
  git clone https://github.com/ibnufachrizal/code/
  ```
1. Masuk ke folder project yang ingin di jalankan
  ```bash
  cd path/to/project_name
  ```
2. Run CMD for windows
3. Convert .pas to .exe
  ```bash
  fpc file_name.pas
  ```
4. Run Pascal Project
  ```bash
  file_name.exe
  ```
